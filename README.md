## Voir mon code:
```
$ git clone https://github.com/Gwekkeo/tab_carousel_js.git
```
```
$ cd tab_carousel_js
```

## Etapes:
* Installation des gems (sans la production)

```
$ bundle install --without production
```
```
$ rails server
```

#### Dans la barre d'url du Navigateur web
```
$ localhost:3000
```
